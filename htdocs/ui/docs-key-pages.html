The Linking-You vocabulary allows you to explicitly state the key pages of your website. Accept for "homepage" for which we recommend using foaf:homepage as it's already very well established.

Note that these terms link an organisation to a document, not a dataset.

Prefixes

@prefix oo: <http://xmlns.com/foaf/0.1/>. @prefix lyou: <http://purl.org/linkingyou/>.

Pattern

<ORG-URI> foaf:homepage <URL> ; lyou:about <URL> ; lyou:events <URL> ; lyou:conferences <URL> ; lyou:undergraduatae <URL> ; lyou:international-students <URL> .

List of Linking You terms

For the full list of terms, please visit http://purl.org/linkingyou/

Just use the terms that are appropraate for your organisation, the are all optional, although homepage is recommended.

Example

<http://www.example.ac.uk/#org> foaf:homepage <http://www.example.ac.uk/> lyou:about <http://www.example.ac.uk/about-us.html> lyou:events <http://www.example.ac.uk/events/> lyou:conferences <http://www.example.ac.uk/conference-and-hospitality/conferences.html> lyou:undergraduate <http://www.example.ac.uk/study-here.html> lyou:international-students <http://www.example.ac.uk/students/international.html> .
